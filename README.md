# Background Remover Tool

This project is a web application for removing backgrounds from images using Streamlit. It allows users to upload images, edit them by adjusting contrast and brightness, and use a magic brush tool to erase or restore parts of the image.

## Features

- Upload images in JPG or PNG format.
- Remove backgrounds from images.
- Adjust contrast and brightness of the edited images.
- Use a magic brush tool to erase or restore parts of the image.
- Download the final edited image in PNG format.

## Installation

To run this project, you need to have Python installed on your machine. Follow these steps to set up the project:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/bgremover.git
   cd bgremover
   ```

2. Create a virtual environment (optional but recommended):
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

To start the application, run the following command:

```
bash start.sh
```

This will launch the Streamlit app in your default web browser.
